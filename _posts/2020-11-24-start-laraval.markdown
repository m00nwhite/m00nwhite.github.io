## 安装
使用composer下载laravel，composer的安装请参考[这里](http://moonwhite.site/php/web/2020/11/23/install-composer.html)
```bash
composer create-project laravel/laravel laravel.test "5.5.*"
```

## 运行
下面命令会在本地8000端口开启一个laravel服务。
```bash
php artisan serve
```
